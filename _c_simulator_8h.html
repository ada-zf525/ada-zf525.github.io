Detailed Description

Mass-balance and economic-performance simulator for encoded circuits.

Wiring contract for the rest of the project:

  • GATypedefs.h owns the circuit-vector encoding.
  • CSimulator owns mass-balance simulation, recycle convergence, stream routing, and economic scoring.
  • circuit_performance(std::span<const int>) is the GA-facing scalar fitness interface required by RequiredFunctions.h.
  • CSimulator::Simulate(...) is the richer diagnostic interface for testing, debugging, benchmarking, and presentation evidence.

The simulator deliberately uses the helper functions in GATypedefs.h instead of manually decoding vector offsets. This keeps vector-layout changes local to one header and avoids hidden wiring bugs.
