◆ seed_rngs()

void seed_rngs ( unsigned seed)

Improved PRNG using mt19937 (Mersenne Twister Algorithm) with one generator per OpenMP thread, using the provided fast_random1 approach.

This approach is an improvement from the old LCG framework, as it covers a much larger period, is thread safe, and is made compatible with the existing RNG usage.
