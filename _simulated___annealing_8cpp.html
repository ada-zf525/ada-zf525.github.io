Run simulated annealing over the variable part of a circuit vector.

Mirrors the contract of optimize_span. Returns the best valid circuit encountered during the annealing trajectory, evaluated against the supplied fitness function. On total failure to seed any valid initial state the returned vector is a marker whose unit count is set to a negative value and whose performance is negative infinity.

Memory ownership of the returned data follows the BaseResult convention and is released by the destructor of the returned object.

Parameters
fixed_prefixFixed circuit-vector prefix that determines the shape of the search space.
performance_functionScalar fitness function evaluated on full circuit vectors. Required only to be defined on validity-passing inputs.
validity_checkerPredicate used to reject ill-formed neighbours.
ga_functionsGenerator, mutator, and extent callbacks that define the variable-part search space. The crossover and similarity entries are ignored.
sa_parametersAnnealing schedule and termination parameters.
Returns
The best circuit vector found together with its fitness, or the failure marker described above.

Run simulated annealing over both unit types and wiring.

Counterpart to optimize_swappable_span. Each unit's type is selected from {Type A, Type B} and may be mutated during the trajectory according to SA_Parameters::type_mutation_probability. When a type is flipped a fresh wiring vector is generated for the new prefix, since the variable-part length depends on the unit-type sequence. otherwise the wiring is mutated in place via GA_Functions::vector_mutator.

Memory ownership of the returned data follows the BaseResult convention and is released by the destructor of the returned object.

Parameters
num_unitsNumber of separation units in the circuit. Must be positive. non-positive values produce a failure marker without invoking the annealer.
performance_functionScalar fitness function evaluated on full circuit vectors.
validity_checkerPredicate used to reject ill-formed neighbours.
ga_functionsGenerator, mutator, and extent callbacks that define the variable-part search space.
sa_parametersAnnealing schedule and termination parameters, including SA_Parameters::type_mutation_probability.
Returns
The best circuit vector found together with its fitness, or a failure marker of the form [1, -1, 3] with performance set to negative infinity.
