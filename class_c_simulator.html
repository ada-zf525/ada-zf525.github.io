Detailed Description

Circuit mass-balance simulator.

This class is intentionally stateless apart from numerical options. A fresh instance can be created cheaply wherever a circuit needs to be evaluated.

mass_balance::SSimulationResult CSimulator::Simulate ( std::span< const int > circuit_span) const
nodiscard

Simulate the steady-state mass balance of an encoded circuit.

Parameters
circuit_spanFull circuit vector using the agreed GATypedefs.h encoding.
Returns
Diagnostic simulation result containing final product streams, convergence information, mass residual, and economic value.

The method uses successive substitution. Each iteration computes every unit output from the previous unit-feed estimate, resets all unit feeds/products, injects the external feed, routes every unit output according to the circuit vector, and checks the relative change in unit feeds.
