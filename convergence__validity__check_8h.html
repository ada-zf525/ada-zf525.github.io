Check whether a circuit is numerically usable after mass-balance simulation.

Parameters
circuit_spanFull encoded circuit vector.
optionsSimulator options controlling fixed-point iteration, tolerances, and failure fitness.
mass_residual_toleranceMaximum value of accepted absolute mass-balance residual in kg/s.
Returns
Detailed convergence validity result containing pass/fail status, convergence diagnostics, economic value, mass residual, and failure reason.

This function runs CSimulator::Simulate() and then applies simulation-level sanity checks. A circuit passes only if the simulator reports a valid shape, the fixed-point mass-balance iteration converges, the economic value is finite, the mass residual is finite and below the requested tolerance, and all final product streams are finite.

Note
This function performs a full simulation. It is intended for fitness-evaluation diagnostics or generator/mutator debugging, not as a high-frequency replacement for the fast topology validity checker.
